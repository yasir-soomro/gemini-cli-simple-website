# GEMINI System Instructions for Real-Estate Website Project

## Project Overview
You are assisting in building a **modern, responsive, animated real-estate website** using HTML and CSS. The website will include a hero section, property listings, search/filter functionality, testimonials, a contact form, and a footer. All code must be clean, semantic, and production-ready.

---

## Behavior Instructions
1. **Code Quality**
   - Write clean, well-indented, and semantic HTML and CSS.
   - Use reusable classes and meaningful naming conventions.
   - Comment key sections for clarity.

2. **Responsiveness**
   - All layouts must be fully responsive (mobile, tablet, desktop).
   - Use CSS Flexbox, Grid, and media queries.

3. **Animations**
   - Implement smooth animations for hover effects, transitions, sliders, and buttons.
   - Use CSS transitions, keyframes, and transforms where appropriate.
   - Ensure animations are subtle and professional.

4. **Interactivity**
   - Buttons, links, and input fields should have hover/focus effects.
   - Carousels/sliders should support both auto-scroll and manual navigation.

5. **File Structure**
   - Keep HTML and CSS files organized and separated logically.
   - Suggest proper naming for folders (e.g., `assets/`, `css/`, `images/`).

6. **Instructions for Gemini CLI**
   - When prompted, generate **full HTML and CSS code** only.
   - Do not include any explanations unless explicitly asked.
   - Include embedded comments in code for structure and animation logic.
   - Make code copy-paste ready for deployment.

7. **Project Focus**
   - Maintain a modern, clean, and professional UI.
   - Ensure all code is optimized for performance and readability.
   - Avoid unnecessary frameworks unless requested.

---

## Example Commands Gemini Might Execute
- `gemini run generate_ui.py` → Generate a new responsive HTML/CSS layout.
- `gemini edit index.html` → Modify hero section or navigation bar.
- `gemini find "property-card"` → Search for property card components in HTML/CSS.
- `gemini save-memory "project-style"` → Save consistent design patterns.

---

## Output Guidelines
- Provide complete, ready-to-use code.
- Include comments for clarity.
- Ensure all responsive and animation features are functional.
- Follow a consistent style across the project.
